# Dados

Adicione o arquivo `vgsales.csv` dentro da pasta `data/raw/`.

Caminho esperado:

```text
data/raw/vgsales.csv
```

O arquivo deve conter as colunas:

- Rank
- Name
- Platform
- Year
- Genre
- Publisher
- NA_Sales
- EU_Sales
- JP_Sales
- Other_Sales
- Global_Sales

Não publique dados sem conferir a licença e a fonte original.
